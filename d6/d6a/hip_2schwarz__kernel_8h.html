Device kernel for schwarz extrude This can probably be done i a better way... We sum the "shell" of a1 that is l1 steps in scaled with f1 with the shell of a2 that is l2 steps in and scale with f3. Right now we just thorw away all arrays that are not on the first face of dimension (nx-2)(nx-2) It should be noted that a1, a2 are often the same array. If l1,l2 are not the same or if one is not 0 this might lead to a race.
